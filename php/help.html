概要说明

            API访问地址
              正式服：http://www.fnying.com/php/

            返回数据格式
              json

            异常返回
              errcode 返回码
              errmsg  失败信息

              常见错误码：
              -1  系统繁忙，稍候再试
              110 服务异常中断，请与管理员联系
              120 参数错误

获取用户UUID

http://www.fnying.com/php/get_uuid.php

            输入参数
              无

            返回
              uuid            UUID
              errcode = 0     请求成功

            说明
              获取用户唯一识别标识

网页访问统计

http://www.fnying.com/php/cnt_action.php

            GET参数
              referrer        来源URL
              url             访问URL
              uuid            UUID

            返回
              统计而已，不管对错，不返回任何东西

            说明
              将来源URL，访问URL的参数剥离后保存入cnt_url_action表

网页访问概要统计刷新

http://www.fnying.com/php/rpt_overview_refresh.php

            参数
              无

            返回
              刷新数据，不管对错，不返回任何东西

            说明
              将rpt_overview表中所有记录读出，按照URL关键字（url_key）统计cnt_url_action表符合条件记录的条数，
              并更新rpt_overview表中对应记录的报告数据（rpt_count）和报告时间（rpt_time）。
